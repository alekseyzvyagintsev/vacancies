# Проект Vacancies

## Описание

Проект **Vacancies** — это приложение для анализа вакансий с платформы HeadHunter (hh.ru). Оно позволяет получать, фильтровать, сортировать и сохранять данные о вакансиях, а также анализировать их через базу данных PostgreSQL.

Приложение предоставляет интерактивный интерфейс для работы с вакансиями по ключевому слову (по умолчанию — "разработчик"), фильтрации по зарплате, просмотру топ-N вакансий и выполнению аналитических запросов.

## Функциональные возможности

- **Получение вакансий с HeadHunter API**: приложение подключается к API hh.ru и получает данные о вакансиях по заданному ключевому слову.
- **Фильтрация и сортировка вакансий**: вакансии фильтруются по валюте (RUR), диапазону зарплат и другим критериям.
- **Сохранение данных в JSON**: полученные вакансии преобразуются в объекты и сохраняются в формате JSON.
- **Работа с базой данных PostgreSQL**:
  - Автоматическое создание базы данных и таблиц.
  - Хранение данных о компаниях (работодателях) и вакансиях.
  - Обеспечение целостности данных через внешние ключи.
- **Аналитические запросы**:
  - Получение списка компаний и количества вакансий у каждой.
  - Получение всех вакансий с указанием работодателя, названия, зарплаты и ссылки.
  - Расчет средней зарплаты по всем вакансиям.
  - Поиск вакансий с зарплатой выше средней.
  - Поиск вакансий по ключевому слову в названии.
- **Интерактивное меню** для выбора аналитических операций.
